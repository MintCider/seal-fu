## ⚠️注意⚠️

本版本依据官方中文版规则书（2026.02.06 版）更新了部分术语翻译，导致人物卡的数据存储格式改变。从旧版本升级而来的话，会出现数据不匹配。可以通过 `.st clr` 清空旧人物卡属性，参考最新术语翻译，重新创建人物卡。

术语变动如下：

| 旧术语     | 新术语     |
| ---------- | ---------- |
| 感知       | 洞察       |
| 状态效果   | 异常状态   |
| 先攻修改值 | 先攻修正值 |

## 功能

- 依据官方中文版规则书（2026.02.06 版），更新术语翻译。
- `.rc` 命令支持在属性名后直接指定骰面数，适用于未录入角色卡的情况（例如：`.rc mig10+mig10-2`）。
